# 🐞 Bug Tracker System (Full Stack SaaS)

A scalable full-stack bug tracking system designed to manage software issues efficiently.  
Built with modern technologies, this system supports multi-user workflows, real-time issue tracking, and secure authentication.

---


---

## 📌 Features

- 🔐 Role-based access control (Admin, Developer, Tester)
- 🐛 Create, assign, update, and track bugs
- 🔄 Real-time issue status management
- 📊 Dashboard for tracking bug progress
- 🔒 Secure authentication using JWT
- 💻 Responsive UI for seamless user experience

---

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS  
**Backend:** Spring Boot, REST APIs  
**Database:** MySQL  
**Authentication:** JWT  

---

## 🏗️ System Architecture

- Modular backend architecture using Spring Boot
- RESTful API design for communication between frontend and backend
- Efficient database schema with optimized queries and indexing

---

## ⚙️ Installation & Setup

1. Clone the repository
    ```bash
    git clone https://github.com/your-username/bug-tracker-saas.git
    cd bug-tracker-saas


2. Backend Setup
    cd backend
    # Configure application.properties (DB credentials)
    mvn spring-boot:run


3. Frontend Setup
    cd frontend
    npm install
    npm start
